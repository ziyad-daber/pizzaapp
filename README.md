# Pizza Recipes App

Une application Android simple qui affiche une liste de recettes de pizza.

## Comment utiliser

1. Clonez le projet.
2. Ouvrez-le dans Android Studio.
3. Lancez l'application sur un émulateur ou un appareil physique.

## Fonctionnalités

* Affiche une liste de pizzas avec des images et des noms.
* Permet de voir les détails d'une recette de pizza en cliquant dessus.
* Inclut les ingrédients, le temps de préparation et les étapes de la recette.
* Écran de démarrage (Splash screen).

## Technologies utilisées

* Java
* Android SDK
* XML pour les layouts
